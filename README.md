# captacion
Prototipo de Sistema de Captación UNIVO Plantel Córdoba

El Sistema está planeado para la asistencia en las visitas a escuelas, ferias y en general para registar los datos generales de los candidados a inscripciones.
El Sistema consta de tres piezas principales:
  1.- El módulo index.html que es el Tablero de Control donde, en tiempo real, se verifica el registro de los datos generales del candidato.
  2.- El módulo registro.html que es el encargado de capturar vía web los datos generales del interesado.
  3.- Una Hoja de Cálculo de GOOGLE, que es donde se registra y almacenan los datos capturados y al mismo tiempo da la información al Tablero de Control.
La versíon al 2 de abril, contempla las observaciones:
* indicar si el candidato cuenta con WhatsApp
* registrar escolarizado o flexible
* validar que el número telefónico capturado sea de 10 dígitos.
